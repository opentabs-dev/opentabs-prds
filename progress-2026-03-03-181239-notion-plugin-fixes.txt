## Codebase Patterns
- Plugin quality checks require running `npm install` first if `node_modules/` is absent (standalone plugins are not part of npm workspaces)
- Biome formatter wraps long throw statements: `if (!x)\n  throw Err(...)` (single-line); multi-arg calls get further wrapped with trailing comma
- ToolError is imported from `@opentabs-dev/plugin-sdk` in all tool files
