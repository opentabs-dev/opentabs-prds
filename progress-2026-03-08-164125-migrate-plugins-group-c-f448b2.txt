## Codebase Patterns
- SDK utilities (getCookie, getLocalStorage, getAuthCache/setAuthCache/clearAuthCache, waitUntil) are exported from `@opentabs-dev/plugin-sdk`
- Auth cache stores typed values directly (no JSON.stringify/JSON.parse needed) — the SDK handles storage as-is
- `waitUntil` rejects on timeout; wrap with `.then(() => true, () => false)` when a boolean return is needed
- Plugin `node_modules` may not be present in worktrees — run `npm install` before `npm run build` if builds fail on missing packages
