## Codebase Patterns
- CLI binary is at `platform/cli/dist/cli.js`, source entry at `platform/cli/src/cli.ts`
- The `--show-config` flag on `start` prints a human-readable display with ANSI color codes (not JSON), so assertions should only check exit code
- The PRD suggested `../../dist/cli.js` for the URL path from src, but the correct relative path from `platform/cli/src/` to `platform/cli/dist/cli.js` is `../dist/cli.js`
- `OPENTABS_CONFIG_DIR` env var isolates tests from real user config at `~/.opentabs/`
- `OPENTABS_TELEMETRY_DISABLED=1` prevents telemetry during tests
- Test files in `platform/cli/src/` are included via `platform/cli/tsconfig.test.json` which has `"include": ["src/**/*.test.ts"]`
