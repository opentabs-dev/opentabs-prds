## Codebase Patterns
- The docs directory (docs/) is a standalone Next.js project with its own package.json and node_modules. Run checks with `cd docs && npm run <check>` not from root.
- docs node_modules are not pre-installed by ralph. Run `npm install` in the docs dir if missing.
- The docs quality checks are: `cd docs && npm run build && npm run type-check && npm run lint && npm run knip && npm run format:check`
- MDX config tables use Table.Row / Table.Cell / Table.Head pattern from the custom MDX components.
