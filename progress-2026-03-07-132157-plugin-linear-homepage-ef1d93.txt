## Codebase Patterns
- Standalone plugins in `plugins/` do NOT have `node_modules/` pre-installed by ralph; must run `npm install` before quality checks
- Plugin properties that override base class members require `override readonly` (e.g., `override readonly homepage`, `override readonly displayName`)
