## Codebase Patterns
- schemas.ts is the central location for all shared Zod schemas, Raw interfaces, and mapper functions in the GitHub plugin
- Tool files import schemas from ./schemas.js, not define their own
- The existing `RawLabel` in schemas.ts (used internally by RawIssue/RawPullRequest) is a simple `{name}` reference type, renamed to `RawLabelRef` to avoid conflict with the full `RawLabel` (with id, color, description) used by label tools
- compare-commits.ts uses its own compareCommitSchema/compareFileSchema which are structurally different from commitSchema — they are NOT duplicates
- Plugin `node_modules/` may not be installed in worktrees; run `npm install` if `npm run build` fails with SDK resolution errors
