## Codebase Patterns
- Alert.stories.tsx uses named const exports (not `export const`) collected at the bottom in a single `export { ... }` statement
- Storybook stories use `render: () => (...)` for stories that need custom JSX; `args: { ... }` for simple prop stories
- The `export default meta` and named exports are always at the bottom of the file
- CVA status variants in Alert override the color/background from the variant (status takes precedence in Tailwind class order)
- Decorative inline SVGs must have `aria-hidden="true"` to satisfy the `lint/a11y/noSvgWithoutTitle` Biome rule
