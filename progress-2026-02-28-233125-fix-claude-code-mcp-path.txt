## Codebase Patterns
- getMcpClientConfigs() in platform/cli/src/commands/start.ts returns config array used by both printMcpClientConfigs() (start.ts) and config.ts; the return type is a union with optional fields — callers destructure only what they need
- printMcpClientConfigs() is exported and called from both start.ts (first-time setup + --show-config) and config.ts (opentabs config show --show-secret)
- Claude Code stores MCP config in ~/.claude.json at top-level `mcpServers` field; the canonical CLI command is `claude mcp add --transport http opentabs <url> --header "Authorization: Bearer <secret>"`
