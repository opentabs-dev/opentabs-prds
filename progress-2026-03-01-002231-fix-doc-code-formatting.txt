## Codebase Patterns
- The docs `.prettierrc` uses: singleQuote: true, printWidth: 120, trailingComma: 'all', arrowParens: 'avoid' — same as the plugin scaffold
- The docs `format:check` only covers `**/*.{ts,tsx,json,md,mjs,css}` — MDX files are NOT checked by Prettier in this project
- All quality checks must be run from inside the `docs/` directory; install deps with `npm install` inside docs if node_modules is missing
- Zod chain collapse rule: collapse `.method().method()...` chains to one line if total line length ≤ 120 chars (including leading indentation)
- Arrow param rule: single identifier params need no parens — `async params =>` not `async (params) =>`, `val =>` not `(val) =>`
