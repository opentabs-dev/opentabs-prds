## Codebase Patterns
- SDK utilities are imported from `@opentabs-dev/plugin-sdk` (getPageGlobal, getMetaContent, waitUntil, etc.)
- `getPageGlobal` uses dot-notation paths (e.g., `'gon.current_username'`) for safe deep property access on globalThis
- `getMetaContent(name)` replaces manual `document.querySelector<HTMLMetaElement>('meta[name="..."]')` patterns
- `waitUntil` replaces manual setInterval polling; wrap with `.then(() => true, () => false)` to return boolean
- Plugin `npm install` may be needed before builds if `node_modules` is missing in the worktree
