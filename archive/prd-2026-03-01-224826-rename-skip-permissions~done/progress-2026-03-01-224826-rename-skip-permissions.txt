## Codebase Patterns
- `platform/mcp-server/src/skip-permissions.ts` — startup-time CLI/env var detection (evaluated once, exported as a function)
- `state.skipPermissions` — combined flag (CLI + env var + config), set at each reload in `reload.ts`
- Config backward compat pattern: read both old and new field names in `parseConfigRecord`, prefer new name; always write with new name on save
- Health endpoint type is defined inline in `http-routes.test.ts` as `HealthResponse` — update this when adding/renaming health fields
