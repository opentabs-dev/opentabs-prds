## Codebase Patterns
- Plugin node_modules are NOT pre-installed in ralph worktrees (only e2e-test plugin is set up). Run `npm ci` in each plugin directory before `npm run build`.
- Plugin dist/ is gitignored — only source files (icon.svg, src/**) are committed.
- The `opentabs-plugin build` step requires `node_modules/@opentabs-dev/plugin-sdk` to resolve SDK version; fails without npm install.
