## Codebase Patterns
- The MCP server's `searchNpmPlugins` in `platform/mcp-server/src/plugin-management.ts` now uses the npm registry HTTP API (`fetch()`) instead of spawning `npm search`. The registry API endpoint is `https://registry.npmjs.org/-/v1/search` with `size=250` and `from` for pagination.
- Direct package probe uses `https://registry.npmjs.org/{packageName}` to check for exact package matches. Response includes `dist-tags.latest` and `versions[latest]._npmUser.name` for publisher info.
- `spawnAsync` is still used by `runNpmGlobal` for install/uninstall/update operations — only search was migrated to fetch().
- AbortController + setTimeout pattern replaces the old Promise.race timeout pattern for fetch-based operations.
