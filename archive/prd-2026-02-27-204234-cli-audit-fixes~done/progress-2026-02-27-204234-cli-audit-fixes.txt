## Codebase Patterns
- CLI source is in platform/cli/src/commands/; tests are co-located as *.test.ts
- node:path's isAbsolute() is platform-native (Windows paths only recognized on Windows)
- Pre-commit hook runs prettier + eslint --fix on staged files automatically
- `import-x/exports-last` rule: do NOT use `export const foo`; use `const foo` and add to the `export { ... }` at end of file
- `import-x/order` with `newlines-between: 'never'`: no blank lines between any import groups; `import type` declarations come after value imports (type group is last in groups array)
- `import-x/first` rule: ALL imports must precede any non-import statements (including vi.mock calls); use Vitest's automatic hoisting of vi.mock
