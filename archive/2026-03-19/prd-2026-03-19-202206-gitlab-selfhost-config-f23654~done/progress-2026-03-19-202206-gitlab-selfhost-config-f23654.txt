## Codebase Patterns
- Plugins with self-hosted support use `configSchema` with a `type: 'url'` field. For plugins with a known SaaS domain (like gitlab.com), keep `urlPatterns` and set `required: false`. For purely self-hosted plugins (like Grafana), use `urlPatterns: []` and `required: true`.
- The `configSchema` must be declared identically in both `src/index.ts` (as a class property with `as const` type assertions) and `package.json` (in the `opentabs` field).
- Plugin `node_modules` may not be present in worktrees — run `npm install` in the plugin directory before building.
