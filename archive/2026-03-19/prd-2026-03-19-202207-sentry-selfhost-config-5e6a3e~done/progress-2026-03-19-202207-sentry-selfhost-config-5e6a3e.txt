## Codebase Patterns
- Plugins are standalone projects in `plugins/` — they have their own `node_modules/`, `package-lock.json`, and are NOT part of root npm workspaces
- Plugin `configSchema` must be declared in both `src/index.ts` (class property) and `package.json` (opentabs field) with matching definitions
- Import `ConfigSchema` type from `@opentabs-dev/plugin-sdk`
- In worktrees, plugin `node_modules/` may not be present — run `npm install` before building
