## Codebase Patterns
- The Slack plugin was previously updated to use `file:../../platform/plugin-sdk` and `file:../../platform/plugin-tools` (commit ceac63f). The e2e-test plugin follows the same pattern.
- Plugin quality checks: `cd plugins/<name> && bun install && bun run build && bun run type-check`
- The installed npm version (0.0.16) does NOT contain the latest SDK utilities (setSessionStorage, putJSON, etc.). These exist only in the local workspace source. Using `file:` references resolves to the local built output.
- In Zod 4, `z.record()` requires two arguments: `z.record(keySchema, valueSchema)`. Using `z.record(z.unknown())` with one arg causes TS2554.
- `deleteJSON` overloaded signature can confuse TypeScript when called with a single argument — pass `undefined` as the second arg for clarity.
