## Codebase Patterns
- When generating template literal code inside an outer template literal, use `\`` for the inner backtick delimiters and `\${varName}` to produce `${varName}` in the generated output
- Biome `useTemplate` lint rule flags string concatenation that could be a template literal — this affects any generated code too
