## Codebase Patterns
- platform/cli/src/commands/doctor.ts already uses spawnProcessSync from @opentabs-dev/shared for all subprocess calls
- The only direct Bun.* reference in doctor.ts is Bun.version, guarded by isBun check (line 59-60)
- spawnProcessSync returns { exitCode, stdout, stderr } (SpawnResult type from @opentabs-dev/shared)
- To mock @opentabs-dev/shared with mock.module, first capture the real module with `await import(...)`, then spread it in the mock factory. Cannot re-import inside the factory (causes circular resolution error for named exports like getEnv)
- Pattern: `const realShared = await import('@opentabs-dev/shared'); await mock.module('@opentabs-dev/shared', () => ({ ...realShared, mockFn }));`
- E2E tests may fail in worktrees that are behind main — merge main to pick up extension fixes before running E2E
