## Codebase Patterns
- `runNpmGlobal` in `platform/mcp-server/src/plugin-management.ts` spawns npm subprocesses with Bun.spawn and uses `proc.exited` to await exit. Always race `proc.exited` against a deadline to prevent hangs.
- The MCP server uses `@opentabs-dev/shared` for platform utilities like `platformExec('npm')`.
- Timer cleanup pattern: capture all `setTimeout` return values and `clearTimeout` them on both success and error paths.
- Fire-and-forget promises (`void someAsync()`) must always have a `.catch()` to prevent unhandled rejections. See `reload.ts` for the canonical pattern.
- String union validation pattern: use a `Set<UnionType>` constant (e.g., `VALID_CONFIRMATION_SCOPES`) and check `.has()` before casting. Fall back to a safe default and `log.warn` on invalid values.
- Array deduplication pattern: before `.push()`, use `.some()` to check for equivalent entries. `===` handles `null` correctly (`null === null` is `true`).
