## Codebase Patterns
- The scaffolder in `platform/cli/src/scaffold.ts` is shared by both `opentabs plugin create` and `create-opentabs-plugin` (via `@opentabs-dev/cli/scaffold` export)
- `generatePackageJson` is a synchronous function that accepts pre-resolved versions as a parameter
- `resolvePluginSdkVersions()` returns a `ResolvedVersions` object with `openTabsVersion`, `zodVersion`, and `source` fields
- The zod peer dependency version always comes from the local SDK (tracks API compatibility), even when the @opentabs-dev version comes from the npm registry
- `platform/create-plugin/src/index.test.ts` has integration tests that run the scaffolder via subprocess
