## Codebase Patterns
- The worker communicates with the dev proxy via IPC using `process.send({ type: string, ... })` and the proxy listens via `child.on('message', ...)`.
- The guard pattern for IPC sends is `process.env.OPENTABS_PROXY === '1' && process.send`.
- `handleConfigSetSkipPermissions` in extension-handlers.ts is the only place where `state.skipPermissions` changes at runtime.
- The existing IPC pattern in index.ts sends `{ type: 'ready', port }` — new message types follow the same structure.
