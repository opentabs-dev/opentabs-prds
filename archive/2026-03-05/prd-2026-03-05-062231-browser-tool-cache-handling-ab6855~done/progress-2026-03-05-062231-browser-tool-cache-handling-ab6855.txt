- Browser tools live in `cache.browserTools`, NOT in `cache.plugins`. When handling `plugin === 'browser'`, always branch to operate on `cache.browserTools` instead of searching `cache.plugins`.

- `reapplyPendingOptimisticUpdates()` in `server-state-cache.ts` already handles both plugin and browser tool pending updates — it re-applies `pendingBrowserToolUpdates` to `cache.browserTools` on every `updateServerStateCache` call.
