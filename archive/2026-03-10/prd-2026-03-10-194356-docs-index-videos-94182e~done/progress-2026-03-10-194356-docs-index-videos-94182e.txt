## Codebase Patterns
- Docs node_modules are NOT pre-installed by ralph setup — run `npm install` in `docs/` before running quality checks
- Docs quality checks: `cd docs && npm run build && npm run type-check && npm run lint && npm run knip && npm run format:check`
- Biome `useSortedClasses` rule enforces Tailwind class ordering — use `npm run lint:fix` or sort manually. Correct order: layout/position → overflow → border → shadow
- Custom MDX components are registered in `docs/components/MDX.tsx` in the `docComponents` object
- Non-illustration components (like YouTubeEmbed) go in `docs/components/` as separate files, NOT in illustrations.tsx
