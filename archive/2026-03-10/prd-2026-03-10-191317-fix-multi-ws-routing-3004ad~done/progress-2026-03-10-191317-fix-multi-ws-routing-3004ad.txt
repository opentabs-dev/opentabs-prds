## Codebase Patterns
- Browser tool descriptions must match the BROWSER_TOOLS_CATALOG in platform/shared/src/generated/browser-tools-catalog.ts. When changing a description, update both the tool definition and the catalog (or run `npm run generate:browser-tools-catalog`). The barrel in browser-tools/index.ts validates at import time and throws on mismatch.
- The `dispatchToExtension` function routes to a single connection via `resolveConnection`. For broadcasting to all connections, use the new `dispatchToAllConnections` helper which dispatches in parallel and collects per-connection results.
- TypeScript strict mode: array indexing returns `T | undefined`. Use `entries()` with destructuring or explicit null checks instead of bare `array[i]`.
