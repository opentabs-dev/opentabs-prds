## Codebase Patterns
- `ensureExtensionInstalled()` in reload.ts runs on every startup/hot reload — it's the right place to detect version changes
- The extension already handles `extension.reload` via message-router.ts dispatch table — reuse existing infrastructure
- `sendSyncFull()` in http-routes.ts runs on every WebSocket connect — chain deferred work after it via `.then()`
- Pre-existing build errors in shared/src/index.ts had stray interface fields outside closing braces — always verify shared types compile
- extension-protocol.ts had duplicate code blocks (stale remnants from previous edits) — always check for and clean up stale duplicates
- platform/plugin-tools/src/commands/build.ts had uncommitted icon validation changes with dead code after `return` — reverted in US-003 by restoring to HEAD state
- When staging pre-existing changes alongside your own, verify the staged version passes lint independently (lint-staged only sees the staged snapshot)
- `ConfigStateResult` in shared already includes `outdatedPlugins` — App.tsx just needs to read it from `fetchConfigState()` result
- Plugins in `plugins/` depend on published npm packages, not local source — features must be published before the e2e-test plugin can use them
- The side panel header area (search bar row) is a natural place for persistent badges — use flex layout with `gap-2` to position alongside the search input
- E2E tests use `platform/browser-extension/` directly (not `~/.opentabs/extension/`) — see `EXTENSION_DIR` in `e2e/fixtures.ts`
- `.tsbuildinfo` files live at `platform/*/*.tsbuildinfo` (package root), not inside `dist/` directories
- `platform/browser-extension/icons/` is gitignored — icons are generated from SVG during build
- macOS ships bash 3.2 — avoid `declare -A` (associative arrays), `mapfile`, and other bash 4+ features in shell scripts

- **Learnings for future iterations:**

  - `${RANGE:+"$RANGE"}` is the portable way to conditionally pass a variable as an argument (expands to nothing when empty, avoids passing an empty string)
