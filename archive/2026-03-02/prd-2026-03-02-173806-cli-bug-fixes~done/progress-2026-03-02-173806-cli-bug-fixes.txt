## Codebase Patterns
- `platformExec('npm')` is the correct way to invoke npm in all spawn calls — resolves to `npm.cmd` on Windows, `npm` elsewhere
- `platformExec` is exported from `@opentabs-dev/shared` and imported in CLI command files
- Prettier enforces trailing commas in multi-element arrays — run `npx prettier --write` to auto-fix formatting errors rather than manually adjusting
- `/health` endpoint returns JSON body with `mode: 'dev' | 'production'` and `x-opentabs-version` response header
- `getPidFilePath` and `parsePidFile` are exported from `platform/cli/src/config.ts` — use these for PID file background detection
- Background server detection: read PID file with `getPidFilePath()`, parse with `parsePidFile()`, check `isProcessAlive(pid)` using `process.kill(pid, 0)`
