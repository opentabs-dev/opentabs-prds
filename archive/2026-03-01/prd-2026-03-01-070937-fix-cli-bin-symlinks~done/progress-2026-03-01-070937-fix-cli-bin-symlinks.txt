## Codebase Patterns
- npm workspaces only create bin symlinks during `npm install` if the target file already exists. For packages with `dist/cli.js` as the bin target, `npm rebuild <pkg>` after build re-links the symlinks.
- `npm rebuild <pkg>` is fast (<1s) even for packages with no native addons — it just re-links bin entries.
