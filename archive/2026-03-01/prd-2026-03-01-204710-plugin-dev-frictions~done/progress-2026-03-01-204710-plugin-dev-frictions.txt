## Codebase Patterns
- Scaffold template is in `platform/cli/src/scaffold.ts` — `generatePackageJson` produces the package.json, `PRETTIERRC_CONTENT` for .prettierrc, `generateReadme` for README.md, `generatePluginIndex` for src/index.ts
- Established plugin conventions (from plugins/slack and plugins/e2e-test): `check` script runs `npm run build && npm run type-check && npm run lint && npm run format:check`; `dev` uses shell background `&` not concurrently; no `concurrently` dependency; `description` at top-level package.json; no `//` comment in opentabs field
- `publishConfig: { access: 'restricted' }` is used by scoped packages (`@opentabs-dev/...`) but NOT included in unscoped scaffold output (scaffold creates unscoped packages: `opentabs-plugin-<name>`)
