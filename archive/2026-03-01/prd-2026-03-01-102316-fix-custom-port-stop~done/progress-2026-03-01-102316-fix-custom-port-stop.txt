## Codebase Patterns
- CLI commands live in `platform/cli/src/commands/`. Key files: `start.ts`, `stop.ts`.
- PID file path comes from `getPidFilePath()` in `platform/shared/src/constants.ts` → `~/.opentabs/server.pid`.
- Port resolution priority: `--port` flag > `OPENTABS_PORT` env > `config.json` port field > `DEFAULT_PORT` (9515). See `platform/cli/src/parse-port.ts`.
- PID file is written with mode `0o600` — always preserve this permission when writing it.
- `parsePidFile` helper added to `stop.ts` to parse both JSON `{"pid":N,"port":N}` and legacy plain-integer formats.
- `overlayscrollbars` / `overlayscrollbars-react` were unused deps in browser-extension — removed.
