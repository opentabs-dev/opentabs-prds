## Codebase Patterns
- Port resolution in CLI commands: use `resolvePort(options)` from `parse-port.ts` — priority: --port flag > OPENTABS_PORT env > config.port > DEFAULT_PORT (9515)
- Commander global options (--port) are propagated via `command.optsWithGlobals()` in action handlers; omitting this causes globals to be missing from options
- `resolvePort` caches the config port; tests must call `resetConfigPortCache()` between tests when varying config file content
