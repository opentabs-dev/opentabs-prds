## Codebase Patterns
- `withDebugger` in `resource-commands.ts` is a general-purpose utility that manages debugger attach/detach; it does NOT disable individual CDP domains — callers that enable domains (e.g., Page.enable) must disable them before the callback returns when reusing an existing session
- `isCapturing(tabId)` from `network-capture.ts` determines whether a network-capture debugger session is already attached; call it inside a `withDebugger` callback to guard `Page.disable` so it only runs on the alreadyAttached path
- ESLint enforces alphabetical import ordering within relative path groups; parent-relative imports (`../`) must be ordered alphabetically among themselves
