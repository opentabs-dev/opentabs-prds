## Codebase Patterns
- The `summary` field goes immediately after `description` in `defineTool()` calls
- The `summary` field is optional in the SDK (`platform/plugin-sdk/src/index.ts:48`)
- Standalone plugins (in `plugins/`) need `npm install` in the worktree before building — `node_modules` may not be present
- Plugin quality checks: `npm run check` runs build, type-check, lint, format:check in sequence
