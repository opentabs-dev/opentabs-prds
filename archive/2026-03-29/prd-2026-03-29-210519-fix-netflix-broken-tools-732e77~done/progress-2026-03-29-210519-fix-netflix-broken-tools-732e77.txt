## Codebase Patterns
- Netflix Apollo cache keys use `Movie:{"videoId":N}` and `Show:{"videoId":N}` format
- `textEvidence` keys in Apollo cache (e.g., `textEvidence({"field":"genres"})`) contain genre tags, NOT synopsis
- Synopsis comes from `contextualSynopsis.text` or `synopsis.value` fields in Apollo cache entries
- `apolloEntryToRawTitle` in schemas.ts is the canonical mapping from Apollo cache entries to RawTitle
- `mapTitle` in schemas.ts maps RawTitle to the output schema — keep field sourcing consistent between mapTitle and get-title-details
- Plugins need `npm install` before building in worktrees (node_modules may not be populated)
