## Codebase Patterns
- Plugin `homepage` field requires the `override` modifier in the plugin class because it's declared as `readonly homepage?: string` in `OpenTabsPlugin` base class (since plugin-sdk 0.0.65).
- When a plugin needs a new SDK feature, update `@opentabs-dev/plugin-sdk` and `@opentabs-dev/plugin-tools` versions together in the plugin's package.json and run `npm install` to update the lockfile.
- Plugins do NOT have `node_modules/` pre-installed in worktrees — you must `npm install` before building when dependencies are missing.
