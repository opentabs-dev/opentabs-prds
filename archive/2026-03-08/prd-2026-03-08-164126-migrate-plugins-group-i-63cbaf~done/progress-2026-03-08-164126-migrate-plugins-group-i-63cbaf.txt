## Codebase Patterns
- Plugin SDK utilities (getCookie, getAuthCache/setAuthCache/clearAuthCache, waitUntil) are imported from `@opentabs-dev/plugin-sdk`
- Auth cache namespace should match the plugin name (e.g., 'vercel' for vercel plugin)
- waitUntil returns a Promise<void> that rejects on timeout; wrap with `.then(() => true, () => false)` for boolean return
- Plugins in worktrees may need `npm install` before building if node_modules is missing
