## Codebase Patterns
- SDK auth cache pattern: `getAuthCache<T>(namespace)`, `setAuthCache(namespace, value)`, `clearAuthCache(namespace)` — replaces manual `globalThis.__openTabs.tokenCache` access
- SDK waitUntil pattern for waitForAuth: `waitUntil(() => isAuthenticated(), { interval: 500, timeout: 5000 }).then(() => true, () => false)` — wraps to return boolean like the original
- Plugins need `npm install` in worktrees before building (node_modules aren't pre-populated for standalone plugins)
- Notion plugin (already migrated) is the reference for this migration pattern
- SDK `getMetaContent(name)` replaces `document.querySelector<HTMLMetaElement>('meta[name="..."]')?.content` — used by GitHub for auth detection
- SDK `getPageGlobal(path)` uses dot-notation for deep property access on globalThis — e.g., `getPageGlobal('__META_DATA__.isLoggedIn')` replaces `(globalThis as Record<string, unknown>).__META_DATA__` casts
- SDK `getCookie(name)` replaces manual `document.cookie.match(...)` patterns — handles URI decoding
- SDK `getLocalStorage(key)` replaces `localStorage.getItem(key)` — returns `string | null` just like the original
- SDK `waitForSelector(selector, { timeout })` replaces custom `waitForElement` polling functions — uses MutationObserver, throws on timeout, takes options object
