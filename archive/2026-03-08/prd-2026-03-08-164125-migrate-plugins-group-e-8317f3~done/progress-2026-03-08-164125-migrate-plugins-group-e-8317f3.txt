## Codebase Patterns
- Plugin SDK utilities (getCookie, getPageGlobal, getAuthCache/setAuthCache/clearAuthCache, waitUntil) are exported from `@opentabs-dev/plugin-sdk` and replace manual cookie parsing, globalThis access, and token persistence patterns
- Auth cache namespace should match the plugin name (e.g., `'sentry'` for the Sentry plugin)
- waitUntil rejects on timeout (throws), so wrap in try/catch and return boolean for backward-compatible waitFor* functions
- Plugin worktrees may need `npm install` before building if node_modules is empty
