## Codebase Patterns
- Plugin SDK exports `getPageGlobal`, `getAuthCache`, `setAuthCache`, `clearAuthCache`, `waitUntil` for common plugin patterns
- `getPageGlobal` uses dot-notation path for safe deep property access on globalThis (e.g., `getPageGlobal('bootstrap.atok')`)
- Auth cache functions use a namespace string (plugin name) to partition `globalThis.__openTabs.tokenCache`
- `waitUntil` returns `Promise<void>` — wrap with `.then(() => true, () => false)` when a boolean return is needed
- Biome formatter prefers single-line arrow functions when they fit within the line width limit
