## Codebase Patterns
- SDK utilities (`getAuthCache`, `setAuthCache`, `getMetaContent`, `waitUntil`) are exported from `@opentabs-dev/plugin-sdk` main entry
- `getAuthCache<T>(namespace)` / `setAuthCache(namespace, value)` use `globalThis.__openTabs.tokenCache[namespace]` under the hood
- `waitUntil` rejects on timeout (throws Error), so wrap with `.then(() => true, () => false)` when a boolean return is needed
- SDK `getMetaContent` uses `CSS.escape(name)` for safety, unlike most local implementations
- Plugins in worktrees may need `npm install` before building (node_modules not always present)
