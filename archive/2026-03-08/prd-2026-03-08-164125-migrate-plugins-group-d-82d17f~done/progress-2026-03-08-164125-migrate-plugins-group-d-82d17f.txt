## Codebase Patterns
- SDK utilities (getCookie, getLocalStorage, getAuthCache/setAuthCache/clearAuthCache, waitUntil) are exported from `@opentabs-dev/plugin-sdk` main entry
- Auth cache namespace should match plugin name (e.g., 'linear')
- waitUntil rejects on timeout (unlike manual setInterval patterns that resolve false), so wrap with `.then(() => true, () => false)` when a boolean return is needed
- Plugin node_modules may not be present in worktrees — run `npm install` in plugin dir before quality checks
