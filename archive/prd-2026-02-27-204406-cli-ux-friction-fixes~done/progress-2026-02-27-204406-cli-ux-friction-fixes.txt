## Codebase Patterns
- CLI commands are in platform/cli/src/commands/. Each command file exports a `register*Command` function.
- The `import-x/order` ESLint rule requires `./` (same-dir) imports before `../` (parent-dir) imports.
- The `import-x/exports-last` rule requires inline `export const` only at end of file; use `export { }` at bottom otherwise.
- Dependencies are NOT pre-installed in the worktree — must run `npm ci` then `npm run build` before quality checks.
- `printMcpClientConfigs` in start.ts builds MCP config blocks for Claude Code, OpenCode, Cursor, Windsurf.
- `DEFAULT_PORT` (9515) is exported from `@opentabs-dev/shared`.
