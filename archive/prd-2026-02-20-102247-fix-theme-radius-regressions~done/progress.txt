## Codebase Patterns
- The docs project uses Tailwind CSS with a theme variable `--radius: 0.5rem` — the `rounded` class maps to this value
- Card-like components (Button, Accordion, Card, Callout, code blocks, file tree, tables) should all include the `rounded` Tailwind class for consistent border-radius
- The `cva` (class-variance-authority) library is used for component variant styling in Button and similar components
- Quality checks for docs: `cd docs && bun run build && bun run type-check && bun run lint && bun run knip`
