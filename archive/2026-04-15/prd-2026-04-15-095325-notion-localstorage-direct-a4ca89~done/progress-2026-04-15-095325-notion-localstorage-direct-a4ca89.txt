## Codebase Patterns
- The notion plugin uses `@opentabs-dev/plugin-sdk` utilities — always use SDK wrappers (getLocalStorage, findLocalStorageEntry, getCookie, etc.) instead of direct browser API access
- `findLocalStorageEntry(predicate)` iterates localStorage safely via iframe fallback and returns `{ key, value } | null` for the first matching entry
- Quality checks for notion plugin: `cd plugins/notion && npm install && npm run build && npm run type-check && npm run lint && npm run format:check`
- The notion plugin node_modules are NOT pre-installed in worktrees — run `npm install` before build/check
