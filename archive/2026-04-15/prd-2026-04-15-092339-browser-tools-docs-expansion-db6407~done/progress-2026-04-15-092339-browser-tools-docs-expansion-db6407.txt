## Codebase Patterns
- docs/ is a standalone subproject (not in npm workspaces). It has no package-lock.json, so use `npm install` not `npm ci` to install dependencies before running checks.
- The docs qualityChecks command runs from within the docs/ directory: `cd docs && npm run build && ...`
- Biome lint shows a schema version info message (not an error) when the schema URL version doesn't match the installed biome CLI version — this is a non-fatal info, exit 0.
