- `FileWatcherEntry` has a `watchers: FSWatcher[]` field â€” access watcher instances from `state.fileWatching.entries[n].watchers[0]` in tests.

- ESLint forbids non-null assertions (`@typescript-eslint/no-non-null-assertion`); use explicit null/undefined guards instead.

- `detect-apis.ts` normalizes WebSocket URLs for comparison using `normalizeWsUrl` helper (wssâ†’https, wsâ†’http). WebSocket endpoints from HTTP upgrade requests use `https://`, but frames report `wss://` â€” always normalize both sides when comparing.
