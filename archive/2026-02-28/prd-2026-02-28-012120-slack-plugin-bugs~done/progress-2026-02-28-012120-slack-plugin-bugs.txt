- Quality checks for plugins/slack: `cd plugins/slack && npm run build && npm run type-check && npm run lint && npm run format:check`

## 2026-02-28 - US-001
- Fixed NaN retryAfterMs bug in `plugins/slack/src/slack-api.ts` lines 397-401
- Root cause: `response.headers.get('Retry-After') ?? 'unknown'` produced a truthy string `'unknown'`, making `parseInt('unknown', 10) * 1000 = NaN` bypass the fallback `60000`
- Fix: read the header as nullable, parse only when non-null, guard with `Number.isNaN`, pass `undefined` when header is absent or non-numeric
- Files changed: `plugins/slack/src/slack-api.ts`
- **Learnings for future iterations:**
