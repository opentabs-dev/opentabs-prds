## Codebase Patterns
- Slack plugin quality checks require `npm install` first — `node_modules` are not pre-installed for standalone plugins
- Standalone plugins (plugins/<name>) run quality checks from within their own directory with their own `npm run` scripts
- The `opentabs-plugin` CLI is a local dependency in each plugin's node_modules — not a global command
