## Codebase Patterns
- NPM_CONFIG_PREFIX controls what `npm root -g` returns — use it to redirect npm auto-discovery to a temp directory in tests
- `startMcpServer` in e2e/fixtures.ts accepts optional `envOverrides` (4th param) to customize env vars for the server subprocess
- Setting an env override to `undefined` deletes that key from the inherited process.env
- Plugin discovery pipeline: resolver.ts (scanGlobalDir) → discovery.ts (discoverPlugins) → loader.ts (loadPlugin)
- `skip-npm-discovery.ts` captures OPENTABS_SKIP_NPM_DISCOVERY at module load time — each spawned server process has its own value
- Plugin name derivation: `opentabs-plugin-foo` → `foo`, `@opentabs-dev/opentabs-plugin-foo` → `foo`, `@myorg/opentabs-plugin-foo` → `myorg-foo`
