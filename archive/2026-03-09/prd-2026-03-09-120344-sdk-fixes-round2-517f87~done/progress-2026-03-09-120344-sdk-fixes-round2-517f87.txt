## Codebase Patterns
- Plugin SDK utilities live in `platform/plugin-sdk/src/` with one file per category (fetch.ts, dom.ts, timing.ts, etc.)
- All SDK exports must be re-exported from `platform/plugin-sdk/src/index.ts` in the appropriate section
- Unit tests for SDK utilities use a local HTTP test server spun up in `beforeAll` with random port
- Fetch utility tests are in `platform/plugin-sdk/src/fetch.test.ts`
- Exports in index.ts are sorted alphabetically within each section
