## Codebase Patterns
- `platform/plugin-tools/src/validate-icon.ts` contains all SVG icon validation: `parseColor`, `isAchromaticColor`, `convertColorToGray`, `generateInactiveIcon`
- `parseColor` and `convertColorToGray` have separate code paths for: legacy comma-separated rgb(), modern space-separated rgb(), hsl/hsla, hex, named colors — each must be updated independently when adding new features
- Prettier formats single-expression arrow functions onto a single line — don't write multi-line arrow functions if ESLint expects single line
- `@typescript-eslint/no-non-null-assertion` is enforced — never use `!` on array accesses; use `?? fallback`
- `platform/plugin-tools/src/commands/build.ts` handles watch mode, plugin registration in `~/.opentabs/config.json`
- `platform/shared/src/index.ts` contains `validateUrlPattern`; tests in `platform/shared/src/validation.test.ts`
