## Codebase Patterns
- CLI commands live in `platform/cli/src/commands/`. Each command file exports a `register*Command` function.
- `toErrorMessage` from `@opentabs-dev/shared` is used throughout the CLI for converting unknown errors to strings.
- `NodeJS.ErrnoException` is the correct type for net.Server errors that carry a `.code` property (EADDRINUSE, EACCES, etc.).
